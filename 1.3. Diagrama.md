# Diagrama de flujo (ASCII) para menu_app.py

A continuación un diagrama de flujo en ASCII que representa la lógica del
programa `menu_app.py`. Está en español y utiliza una estructura simple para
hacerlo legible en la terminal o en editores de texto.

```
                +--------------------------------+
                |  Inicio: mostrar bienvenida    |
                +---------------+----------------+
                                |
                                v
                +---------------+----------------+
                | Mostrar menú (opciones 0-4)    |
                +---------------+----------------+
                                |
                                v
                +---------------+----------------+
                | Leer opción del usuario        |
                +---------------+----------------+
                                |
           entrada inválida? | no              | sí
                      +------+-----------------+-------+
                      |                                v
                      |                    +-----------+-----------+
                      |                    | Imprimir "Entrada      |
                      |                    | no válida" y pedir de  |
                      |                    | nuevo                  |
                      |                    +-----------+-----------+
                      |                                |
                      |                                v
                      |                    (volver a Mostrar menú)
                      v
    +-----------------+------------------------------+
    | Opción == 0? (salir)                             |
    +----------+--------------------+------------------+
               |                    |                  
              sí                    no
               |                    |
               v                    v
  +------------+--------+   +-------+------------------------+
  | Imprimir mensaje de |   | Opción == 1? mostrar archivo  |
  | salida y terminar   |   | con el problema              |
  +---------------------+   +---------------+--------------+
                                       | sí           | no
                                       v              v
                         +-------------+----+    +----+----------------+
                         | Mostrar archivo   |    | Opción == 2? mostrar|
                         | con el problema   |    | dataset             |
                         +-------------------+    +---------+----------+
                                                         | sí     | no
                                                         v        v
                                                   +-----+----+  +--+-------------+
                                                   | Mostrar  |  | Opción == 3?    |
                                                   | dataset  |  | mostrar algoritmo|
                                                   +----------+  +--------+--------+
                                                                       | sí   | no
                                                                       v      v
                                                                 +-----+-----+ +--+--------------+
                                                                 | Mostrar   | | Opción == 4?     |
                                                                 | algoritmo | | mostrar .py file |
                                                                 +-----------+ +--------+--------+
                                                                                 | sí   | no
                                                                                 v      v
                                                                          +------+-----+ +--------------+
                                                                          | Mostrar .py | | Opción fuera   |
                                                                          | ejemplo     | | de rango       |
                                                                          +------------+ +------+---------+
                                                                                                  |
                                                                                                  v
                                                                                       +----------+----------+
                                                                                       | Volver a Mostrar    |
                                                                                       | menú (bucle)        |
                                                                                       +---------------------+
```

Notas:
- Si el usuario escribe `q`, `quit` o `salir`, el programa trata eso como
  la opción 0 y termina.
- Si la entrada no se puede convertir a entero, se imprime "Entrada no válida"
  y se vuelve a solicitar la opción.
- El flujo siempre vuelve al menú principal tras ejecutar una acción (salvo
  cuando la opción es 0 que termina el programa).
