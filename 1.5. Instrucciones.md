FASE 2: Desarrollar el menú (Gemini)

Con los datos obtenidos, necesito ayuda para crear un menú interactivo básico en Python para interactuar con el usuario este debe contar de 5 puntos:
0.	Salir
1.	[nombre pendiente a definir según el desarrollo].
2.	[nombre pendiente a definir según el desarrollo].
3.	[nombre pendiente a definir según el desarrollo].
4.	[nombre pendiente a definir según el desarrollo].

Si el usuario selecciona la opción 1 este beberá desplegar lo siguiente:
•	Desarrollo: Tema, problema y solución
Si el usuario selecciona la opción 2 este beberá desplegar lo siguiente:
•	Desarrollo: Dataset de referencia: fuente, definición, estructura, tipos y escala
Si el usuario selecciona la opción 3 este beberá desplegar lo siguiente:
•	Desarrollo: Información, pasos, pseudocódigo (Python)
Si el usuario selecciona la opción 4 este beberá desplegar lo siguiente:
•	Desarrollo: diagrama del programa (ASCII)
Si el usuario selecciona la opción 0 este beberá desplegar lo siguiente:
•	Desarrollo: [Mensaje de agradecimiento por definir según el desarrollo]
Cada ves que muestres el desarrollo, a continuación, deberás agregar el menú para que el usuario selecciona otra opción, hasta que seleccione la opción 5 salir. Ejemplo el usuario selecciona la opción 1 del menú, se debe mostrar el desarrollo y el menú nuevamente, si el usuario selecciona la opción 2 del menú, se debe mostrar el desarrollo y el menú nuevamente, esto se debe repetir hasta la opción 4. Cuando el usuario seleccione la opción 5 se mostrará el desarrollo de dicha opción que será un mensaje de agradecimiento y se dejara de mostrar el menú.
Importante: no usar ninguna librería solo lenguaje Python natural. Agrega iconos para hacerlo mas llamativo.
